This window has a single button, for re-connecting to the Magellan.

On startup or during operation of the NanoManipulator, it is possible the NanoManipulator will loose communication with the Magellan for some reason. If this occurs, you should see a dialog like this:

If you want to try to reconnect to the Magellan, press the Reconnect to Magellan button.

It is important not to touch the Magellan puck or buttons during the reset process, as this can cause the reset process to fail.

You should hear a double beep from the Magellan, followed by another beep. This last beep signifies a successful connection, and you can begin using the Magellan again.

If you hear no beeps or a long series of beeps, possibly followed by another error dialog box, the NanoManipulator is having difficulty re-establishing communication. Check to make sure the Magellan is plugged tightly into the COM 1 port of the NanoManipulator computer. Make sure there are no kinks in the Magellan cable. If it fails to reconnect again, shutdown and restart the computer, then restart the NanoManipulator. If this fails, please contact customer support at support@3rdtech.com.

If you need to disconnect the Magellan temporarily from the NanoManipulator, see the Environment section for instructions on avoiding the error dialog displayed above.

See the Magellan Controls section for more information on how the Magellan works.
