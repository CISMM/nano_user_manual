The Stripchart graphs data collected during a modification. It can display the data collected versus time or a spatial dimension, and allows you to easily change the range of the data collected. It will "slide" its view of the data so that the most recent data is always displayed. It is designed to provide instant feedback during and after a modification, so you can decide whether the modification went as planned.

For more powerful graphing needs, the modification data can be exported using the File .. Save Modification Data... command.

By default, the Stripchart shows both Topography and Lateral Force, displayed versus the arc-length of the modification so far. This window can be resized to make the graph bigger. To change these defaults, click the Axis config button. This dialog appears:

[Click to see a full-sized image]

The first box in the Axis config dialog contains controls for the x-axis. You can choose to graph data versus time, arch length in nanometers, or the X, Y or Z coordinate where the data was collected. Graphing versus the Z coordinate is especially useful with force-curve modifications. 

The Range box controls how much data is visible on the X axis. If the X axis is set to Time and the Range box is 10, the Stripchart will display the most recent 10 seconds of data. If the X axis is set to Arc length and the Range box is set to 1000, the Stripchart will display data along the last 1000 nm of path the AFM tip has traveled.

The Max pts box controls the maximum number of points per channel displayed in the Stripchart. We have found that the Stripchart can affect the interactivity of the NanoManipulator if it displays more than a few thousand points. If you are doing a modification using the Freehand tool, the Constrained freehand tool, or the Slow line tool, set the Max pts to be 1000 or less. Also, if you are replaying a stream file with several modifications at high speed, reduce the number of points to speed up replay and avoid large pauses.

The Use every nth pt box allows you to skip some data points in order to display a greater range of data. Even though you can display only a few thousand points at interactive speeds while using certain tools, if you display every 10 or 20 data points, you can still watch the history of a long modification while it is happening. This control is particularly useful when using the Sweep style. Since the sweep style collects a data point at the left, center and right of the sweep width each time it steps forward, the Stripchart graph can become very confused. But if you display only every 3rd point, only one point per step is displayed, and the graph is much clearer.

After the first modification takes place, controls are added for each kind of point data collected during the modification. It also adds controls for the Surface X, Y or Z Axis, which allow you to graph the X, Y or Z position of the tip during the modification. 

The Active toggle controls whether this type of data is graphed. Lateral Force and Topography are active by default. 

The Invert button flips the data. This is particularly useful for Lateral Force, which will sometimes be increasing in force as the signal, measured in nA, is going down. 

The Use Y axis button make this data set take over the Y axis. It's scale is displayed on the Y axis.

The Min and Max boxes control the range displayed. If they are empty, the data will auto-scale so that the minimum and maximum values are just visible on the graph. If you type in a min (or max), the minimum (or maximum) data value displayed is locked to that value, no matter what range the actual data values are in. This can be helpful to cut off an outlying data value and look more closely at a particular data range.
