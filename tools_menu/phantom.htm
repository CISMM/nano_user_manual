To change Phantom settings, choose Tools .. Phantom from the menu. This dialog appears:

Reset Phantom

Occasionally the auto-calibration procedure of the Phantom Desktop will not work correctly, and you will not be able to reach the surface with the Phantom after the Center button has been pressed. If this is the case, press the Reset Phantom button. It will cause the NanoManipulator to re-initialize the connection to the Phantom, and usually corrects the problem.

You can also use this button if the Phantom stops responding for some reason. It will require this if the user presses too hard into the surface, and the Phantom tries to display a force greater than the maximum safe value. The hand icon will stop moving if this happens. Hitting the Reset Phantom button will let it respond again. 

Phantom Button

The Phantom button can have two behaviors. The Press and Hold choice requires that you hold the Phantom button down to "activate" a mode. For example, when using Grab mode, the Phantom button must remain pressed down to move the surface view.

We have found that the Desktop Phantom button to be somewhat stiff, and hard to hold down for long periods of time. The Toggle choice allows the Phantom button to "activate" with a press-and-release, and "deactivate" with a press-and-release. For example, when using Grab mode, the first press-and-release will grab the surface, and the surface view will move with the Phantom until the next press-and-release. However, be careful when using Toggle with Touch or Touch Stored modes, because causing the Phantom to display a force for a long time can cause it to overheat. 

Spring Constant

The surface is modeled with a spring, and this constant controls how stiff that spring is. If the value is low, the surface is softer, and a higher value makes a stiffer surface. If the surface is too stiff, sometimes the force the Phantom displays will oscillate, resulting in a "buzz". Normally the default value of 0.5 is desirable.

See the Phantom Modes section for more information on how the Phantom works.
