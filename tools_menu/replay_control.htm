Automatic Lab Notebooks, or Stream files, are designed to allow a scientist to review an entire experiment. A stream file contains data collected when a NanoManipulator experiment was recorded, which allows the experiment to be essentially “replayed.”  (This is useful if, for example, a physicist makes an exciting discovery and would like to look for details missed when the experiment was first performed, or would like to show what happened to other people.)

By default, when an experiment is replayed, it is replayed in real time. Changing the replay speed can slow down or speed up the rate at which the experiment is viewed.  

It may also be useful to jump to a particular point in the experiment, and find out what was happening at that moment. For example, if you know that something interesting occurred at time 2000, then you could go directly to that point in time, and skip the first part of the experiment. If you wish to start the experiment over, you can rewind the stream file. (This is the same as jumping to time zero.)  Both of these functions will be discussed next.

Finally, stream file replay may be either paused or resumed.

Time is measured in seconds since the experiment started. The current time, the replay speed, and whether replay is paused or playing is displayed in the upper left corner of the Surface View Window.

Replay Speed

The replay speed controls how quickly an experiment is replayed. When the replay speed is set to 1, the experiment is replayed in real time. Setting the replay speed to a higher value increases the rate at which you can view the experiment. (Increasing the replay speed to 5, for example, will cause the experiment to be replayed at 5 times the original speed.)

To change the replay speed, first choose Replay Control from the Tools menu. This will bring up the Replay Controls window:

Click in the entry box labeled Speed (1,30) and backspace to delete the old number. The box will turn pink.

Type the new replay speed value, and press enter. The box will turn white, and the new replay speed will take effect.

The replay speed can be set to any value, but the fastest speed the NanoManipulator can keep up with is about 100 times the original speed. 

Jumping to a Time in the Stream File

If you know that interesting things happen starting at a certain time during an experiment, you may wish to jump directly to that time. (This can be much better than waiting for the stream file replay to reach that time.)

Time is measured in seconds since the start of the experiment. 

To jump to a specific time in the stream file, type a time (in seconds) into the box labeled Go to time: and press enter. You will move to the new time in the stream file. The new time can be before or after the current time, which is displayed in the upper left corner of the Surface View Window.

Rewinding the Stream File

Rewinding the stream file starts over the replay of the experiment. To rewind the stream file, click on the button labeled Rewind. This will start the experiment that you are replaying over at time 0.

This is equivalent to jumping to time 0 in the experiment. (See “Jumping to a Time in the Stream File.”)

Pausing the Stream File

Once you reach an interesting place in the stream file, you might want to pause the replay of the experiment. This is something you might do for example if you want to examine the surface at this instant, or to capture a particular image. To pause the replay of a stream file, click on the button labeled Pause. To resume the replay of the stream file, click on the button labeled Play.
