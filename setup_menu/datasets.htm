This dialog allows you to choose what data to collect with your SPM. Choose Setup .. Datasets from the menu to open this dialog:

The first column controls what scan data is collected. The second column controls what data is collected during a modification. The third column controls what data is collected when a force curve is taken.

If a data set is chosen which is not possible to collect at the same time as the other data sets being collected, the check button will turn red when you choose it, and then turn back off after a moment. 

Note: This dialog is not available when you are viewing stream files or static data files. 

Number of Samples

The modification data sets have an additional column of numbers in entry widgets, which allow you to set how many point samples are averaged by the SPM software before the data is returned to the NanoManipulator. Averaging is used to reduce the amount of noise in the measurement, but it increases the time required to collect data at each location. Typical values for most data sets are 10 samples averaged, but the default for Topography is 90 samples, because it benefits from more averaging.

Reducing the number of samples and the number of different types of data collected will increase the speed with which the tip can follow the Phantom's movements. If you are doing a delicate modification with the freehand tool, this will improve the touch response during modification.
