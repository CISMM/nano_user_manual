Height Plane

The height plane is the surface displayed in the Surface View Window. Choose Height Plane from the Setup menu, and choose any available plane from the list:

You cannot choose "none" as the height field - if you do, Topography-Forward will be automatically selected. If you wish to display a completely flat surface, set the Z scale to zero, as described in the next section.

When you open the NanoManipulator, it creates a plane called "Empty-HeightPlane". This plane remains available if you open a static data file, but contains no data. It will be removed if you open an SPM connection or stream file. 

When you are connected to an SPM, and therefore in all stream files, the plane named Topography-Forward is forced to exist. However, if you use the Datasets dialog or the ThermoMicroscopes software to change the scan data sets so the SPM is not collecting Topography-Forward, this plane of data will be "empty"; all it's data values will be zero. Use the Height Plane control to choose another scan data set.

Note: if you are connected to an SPM and you wish to use Touch to modify the surface, the Height Plane must be the same as one of the Datasets collected during a modification. Otherwise, you won't be able to feel the sample surface. 

Z Scale

The z-scale controls the ratio of the image’s height to its length and width.  Because it can exaggerate height, adjusting the z-scale can make certain features more obvious. 

To modify the z-scale, choose Height Plane from the Setup menu. Type a new scale factor in the Z scale entry box. Because the surface will often move out of view , the surface will center when the z-scale is changed.

Be careful using this tool when you are studying the shape of features on your sample. Because this is a non-uniform scaling, the exaggeration of the height of features on the surface can be misleading.
