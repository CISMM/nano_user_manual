Contour lines show the values in a dataset by displaying iso-lines at different data values. Contour lines are traditionally used in terrain maps to show elevation. 

Choose a Contour plane from the pop-up list. Its data values will be shown using contour lines. To turn off contour lines, choose "none" as the Contour plane. 

Every tenth contour line is white, but the color of other contour lines can be set using the Set contour color button, which brings up the standard color-choice dialog.

The Line width controls what percentage of the space the contour lines occupy - between 1 and 100 percent

The Line spacing controls how far apart the contour lines are. This number is in the same units as the dataset you are displaying; nanometers for topography, but nano-amps for lateral force. It is automatically set to show several contour lines over the range of data when you choose a new Contour plane. 

The Calculate Data Planes button allows easy access to the Calculate Data Planes dialog in the Analysis Menu. It is often desirable to create a flatten plane to display as contour lines, because the sample substrate often has a slope which is difficult to eliminate by positioning the sample under the SPM scanner. Instead, you can have the data automatically leveled using a flatten plane.
