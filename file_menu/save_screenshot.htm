Saving the screen image allows you to store the image displayed in the Surface View window. Images may be saved in either tiff or pnm formats.

To save the screen image, first choose Save Screen from the File menu. This dialog appears:

If you want the text you see on the screen to appear in your final image, then you can leave the box Show screen text marked. If you do not want the screen text to appear in your final image, then you should unmark the Show screen text box by clicking on it. The only item visible will be the surface, which makes it easy to add labels or annotations using an image editing software package. 

You can also decide which file format you wish to use. Choose either TIFF (Tagged Image File Format) or PPM/PGM (Portable Pixel Map/Portable Gray Map) from the pop-up list. 

Once you are satisfied with your settings, click the Save button to save your screen image, or click the Cancel button if you no longer wish to save the image.

Now choose the appropriate directory and file name, and click the Save button. Don't forget to make sure your new filename has the .tif or .ppm extension.

When you click Save, the Surface View window will come "forward", so that no other windows are obscuring it. This is necessary to capture a clean picture of what is in the window.
