To open a plane of data, choose Open Static File from the File menu.

A standard open file dialog appears:

Choose any file that contains scan data. The data must be contained in a square grid. Several formats are understood by the NanoManipulator:

ThermoMicroscopes - from the SPMlab software, both version 4 and 5.
PPM- a standard image file, "Portable Pixel Map", where the data values are represented by shades of gray.
Text - This format includes a short header specifying the dimensions of the grid and then contains a list of data values in text format.
SPIP - Image analysis program file format.
UNCA Image - an image format for use with UNC-CH's freely available /usr/Img package for unix platforms.

The file extension is ignored - the NanoManipulator attempts to determine what format the file is by the first few lines of information contained in the file.

There are some restrictions on which static data files can be opened at any one time. If you have already opened a static data file, a stream file or an SPM connection, the new static data file which you open must be of the same grid size and be collected over the same region as the currently open data. This allows you to open SPMLab files which contain data collected at the same time, like topography and lateral force, or files from two successive scans of the same region. You cannot open SPMLab files at the same time if they are of different regions - this prevents misinterpretation of the data, because the NanoManipulator only maintains one scan region, so the second data file would be scaled incorrectly. If you need to open this second data file, choose File .. Close first.
