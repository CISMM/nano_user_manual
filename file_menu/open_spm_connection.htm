To open a connection to an SPM, choose Open SPM Connection from the File menu. 

The Open SPM Connection dialog appears:

First, choose which SPM to connect to. The NanoManipulator will be pre-configured with the list of SPMs at your site. (The Installation Guide has information on changing this list if this is not the case.)

Next choose the name and location of an automatic lab notebook file, also called a stream file. Click the Choose... button to use a standard file save dialog to select the directory and type in a filename. 

The NanoManipulator assumes that you will use the ".nms" extension for the file, and will append it to any name you specify which does not end in ".nms". Do not choose a file that already exists; the NanoManipulator won't overwrite lab notebook files. 

Today's date is also included in the suggested file name, as an easy way to keep track of when you recorded lab notebook files. We suggest changing word "log" to the name of your sample. Then click Save.

The Open SPM Connection dialog is again displayed, with the name of the lab notebook file displayed. When you hit Open, the NanoManipulator will attempt to open the lab notebook file you specified, and to connect to the ThermoMicroscopes SPM you chose. If there is a problem, an message will be displayed and you will be asked to specify a different file or directory.
