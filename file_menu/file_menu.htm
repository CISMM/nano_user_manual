The File menu provides commands to open and save files, and exit.

The NanoManipulator allows you to open and save several forms of files and data. 

Open Static File
Open Stream File
Open SPM Connection
Close
Save Screen
Save Plane Data
Save Modification Data

Exit closes the NanoManipulator program and all NanoManipulator windows. If you were connected to an SPM and created a stream file, the stream file is saved to disk when you exit.
