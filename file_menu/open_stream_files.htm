To open an automatic lab notebook, choose Open Stream File from the File menu. These files are called stream files because they store the stream of data produced by the SPM during an experiment. 

A standard open file dialog appears:

The standard extension for these files is ".nms", short for NanoManipulator Stream. Choose any automatic lab notebook file to open, and it will start playing back. See Replay Control for information on how to control automatic lab notebook files.
