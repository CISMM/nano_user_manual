Saving plane data allows you to export scan data for analysis or display in another program, including the ThermoMicroscopes' SPMlab software. 

To save plane data, first choose Save Plane Data from the File menu.

This dialog appears:

Choose which data to save from the Plane to extract data from: pop-up list. This list include all available data layers. If you are connected to an SPM, it contains all scan data sets you are collecting, plus any calculated data planes you have created. For a stream file, it contains the scan data sets that were collected when the experiment was recorded, plus any calculated data sets you have created. It also contains the names of any static files that you have loaded. Any of these data layers can be selected.

When you select a data layer, the Format for saved data: pop-up list changes to show you which formats this data can be saved in. The usual list is:

ThermoMicroscopes - can be imported by the SPMLab software. 
PPM- a standard image file, "Portable Pixel Map", where the data values are represented by shades of gray.
TIFF - another standard image file, which can be imported by most programs. 
Text (MathCAD) - Text format that is easily imported by the MathCAD program.
SPIP - Image analysis program file format.
UNCA Image - an image format for use with UNC-CH's freely available /usr/Img package for unix platforms.

After you've chosen which plane and format to use, click Save, and this standard file dialog appears:

Choose a location and filename, and click Save.
