Mouse Phantom Emulation

By click-and-drag in the Surface View window, you can move the hand icon. In the NanoManipulator, the hand icon represents the position and function of the Phantom, as explained on the Phantom Modes page. In NMViewer, or if a Phantom is not connected to the NanoManipulator computer for some reason, the hand icon is moved indirectly by the mouse. These actions affect the hand-icon:

  1. Drag with the left mouse button to translate the hand icon. Up-down and left-right motion of the mouse move the hand-icon in corresponding ways. 
  2. Drag with the right mouse button to translate the hand icon, as if the Phantom button was also held down. For the remaining actions, either the left or right mouse button may be used. 
  3. If you also hold down the Shift key, dragging up and down will translate the hand icon in-out instead.
  4. If you hold down the Ctrl key, the hand icon will rotate. Dragging left-right will rotate around a vertical axis, and dragging up-down will rotate around a horizontal axis. 
  5. If you hold down Ctrl and Shift, dragging left-right will rotate around an in-out axis.

These complicated controls are necessary to emulate the full flexibility of the Phantom. The Phantom has 6 "degrees of freedom", which means it can translate in 3 dimensions, and also rotate in all 3 dimensions. The mouse is limited to translation in 2 dimensions. By using the Shift and Ctrl keys, the mouse can achieve 6 degrees of freedom.

Mouse Phantom is a new entry in the Tools menu, which has a few controls for changing the behavior of the mouse.

If you choose Tools .. Mouse Phantom, this dialog appears:

Reset Mouse Phantom puts the hand-icon in the center of the screen again.

If the hand-icon moves too slowly when you drag the mouse, you can increase the Translational Motion Scale or Rotational Motion Scale to speed it up. If the hand-icon moves too fast, decrease these numbers. Drag the slider to change the numbers.

The text at the bottom provides a reminder for how to operate the Mouse Phantom emulation.
