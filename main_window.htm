The main window is named "NanoManipulator". 

It contains controls for the tasks performed while using the NanoManipulator: 

Choosing whether to start a new experiment or review an old one.
Setting up which data to collect, and how to display that data.
Controlling the SPM, both during imaging and modification.
Analyzing data, through examination and calculation.
Manipulating the view and characteristics of the virtual environment window, which contains the 3 dimensional view of the sample.

Menus

Each menu is explained on its own page.

File Menu - Open new or recorded experiments, and save data.

Setup Menu - Choose which data to collect, and how to display it. 

Tip Control Menu - Control the SPM's imaging and modification parameters.

Analysis Menu - Calculate new data from existing data, or apply tools to examine your data.

Tools Menu - Additional tools, including controls for replaying a saved experiment.

Buttons

Grab, Center, Position Light ...

This set of buttons allows you to easily control what you use the Phantom to do. They are described in the Phantom Modes page. They can be selected by clicking on them with the mouse, or by hitting the Magellan button indicated in the enclosing box. See the Magellan Controls page for more on using the Magellan. 

Immediate controls

This Detail/Speed control allows you to reduce the complexity of the surface the NanoManipulator is displaying, so that Phantom interactions will be smoother. If you collect more than 300x300 data points, the NanoManipulator may slow down, so that the motions of the hand icon on the screen lag behind the motions you make with the Phantom. If this occurs, click on a diamond closer to the More Speed end of the scale. The surface detail will be reduced, but the NanoManipulator will respond more quickly, making it easier to perform tasks with the Phantom, especially manipulations. Note that all the scan data is still being collected, it is just not all being displayed. You can always click on the diamond next to More Detail to see all of your data at a point when you don't need to use the Phantom.

The Stop/Start Scan button allows you to pause and resume scanning the surface, just like the scan button on the toolbar of the ThermoMicroscopes software.

The Withdraw Tip button allows you to withdraw the SPM tip, just like the withdraw tip button on the toolbar of the ThermoMicroscopes software. This is a one-way button - to re-engage the tip, you will need to use the ThermoMicroscopes software.
