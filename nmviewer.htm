The NMViewer is another application that comes with the purchase of the NanoManipulator. It is a viewer for NanoManipulator automatic lab notebook files and for static data files.

It can be installed on any computer in your facility, using the provided install program. It requires a computer with a graphics card that supports OpenGL. It will run best if the graphics card uses hardware to accelerate OpenGL, but this is not required. We have tested it on computers running Windows 98, Windows NT, and Windows 2000, and we expect it will run on computers with Windows 95 and Windows ME.

It includes all the controls from the NanoManipulator application, except those that control a live SPM or control the Phantom. 

Run setup.exe inside the NMViewer folder on your NanoManipulator software CD to install NMViewer. 

Follow the Visualization and Replay tutorial for information on how to access the features of the NMViewer. 

Since the Phantom is not available when running the NMViewer, the mouse is used instead to move the position of the hand-icon, as explained on the Mouse Phantom page.
