Flattened Plane

Flattening the height plane eliminates the slope in the surface.  (For example, if the original sample was placed at an angle, so that it was higher at one end than at the other, then flattening the height plane tries to “undo” this positioning – causing the height plane to again be level, or “flat.”)

The flatten plane tool uses the position of the Measure Lines to determine the three points to use as a plane. Move the three measure lines so they intersect the surface on the substrate - and not on objects or debris that is higher than the substrate. 

Choose Analysis .. Calculate Data Planes to bring up this dialog:

Click in the Flatten plane box and type the name of your new, “flattened” data. When you hit Enter, the new flatten plane will be created. It can be used for a color map or contour line display. 

When connected to an SPM, it is not recommended that you use a flatten plane (or any calculated plane) as the Height Plane, because you will not be able to touch the surface.

Line-by-line Flattened Plane

This method of flattening data eliminates differences between scan-lines. Occasionally the scan-lines from the SPM will be offset relative to each other. Line-by-line flattening applies an offset to every scan-line except the first so that its average data value is the same as that of the first scan line. 

Click in the Line-by-line flatten plane box and type the name of the new plane to be created. When you hit Enter, the new line-by-line flatten plane will be created. It can be used for a color map or contour line display. You can also use the line-by-line flatten plane as the Height Plane for presentation or analysis, but don't use it as the height plane when connected to an SPM, or you won't be able to touch the surface.

Sum Plane

These controls allow you to calculate data in a new plane based on the data in two other planes. This can be useful when comparing two planes of data, or when analyzing error signals. Each data point in the sum plane is calculated like this

sum plane point = (first plane point) + ((scale factor) * (second plane point))

Choose the first and second planes from their pop-up lists, and specify a scale factor. The default scale of -1 will simply subtract the data in the second plane from the data in the first plane. Finally, type in a name for the new sum plane and hit Enter. The new plane will be created with the calculated data.

Dynamic Data

All these methods for creating a calculated plane are dynamic - they create a plane which is linked to its source plane(s), so any new data that comes into the source plane(s) from a stream file or a live connection to an SPM will cause the calculated plane to update as well.

Calculated Planes and Performance

Each calculated plane takes some time to update as new data comes from an SPM or from a stream file. You may notice a performance slow-down if you create several calculated planes. If this becomes unacceptable, choose File .. Close. This will save your stream file if you are connected to an SPM, and clear out all the calculated planes you have created. Then re-open the SPM connection with a new stream file, or re-open the stream file you were viewing.
