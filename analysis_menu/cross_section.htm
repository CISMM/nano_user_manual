The Cross Section tool allows you to make measurements of data along lines.

Type controls whether the cross section will rotate freely, or be constrained so that it lies either parallel to the scan edges, or at 45 degrees to the scan edge. Rotation is controlled by the twist of the Phantom pen.

Width determines whether the cross section expands to touch the edges of the scan, or can be set manually. If the Width control is set to Variable, you can drag the ends of the cross section markers to set the width.

The Reset button clears a cross section - its graphs and data are cleared, and its control marker is cleared. The next time you click and it is not near the ends or middle of the other cross section, it will create this cross section again in the new spot.

The Save button saves the data from a cross section in a text file, for loading into another graphing or analysis software package. (UNDER DEVELOPMENT)

After a cross section is placed, additional controls for it appear for each plane of data that is currently open. Each control affects only a single data layer for one cross section, giving you maximum flexibility in how data is displayed.

Active turns a data source's graph on and off.

Inv inverts a graph, which is especially useful for lateral force data.

Use Y allows a data layer to use the Y axis of the graph. The first cross section uses the left Y axis, and the second cross section uses the right Y axis.

The Min and Max entry boxes control the range of data displayed. Leave them blank to have the graph automatically expand to the minimum and maximum data value.

In the graph itself, red, yellow and blue lines can be dragged across the graph. These pick particular data points to be displayed in the Cross Section Data window:

XY Distance lists the separation in nanometers between the pairs of lines

Then, for each plane of data currently available, the absolute value at each line is displayed, along with the difference between pairs of lines. This gives a quick way to measure meaningful values from the data you are examining.

To place a cross section with the Width control set to Full, simply click with the Phantom pen. The cross section will be oriented along the scan line, similar to the red cross section show. Move the green aim line close to the center vertical line of the cross section, and it will highlight by turning purple. Then you can click and drag the cross section to change its position, and twist the Phantom pen to change its orientation.

To place a cross section with the Width control set to Variable, click and drag with the Phantom pen. The center will be placed where you clicked, and the width and orientation will be determined by where you drag. After it is placed, you can change the position and orientation by dragging the center line, or change its width and orientation by dragging either end line.
