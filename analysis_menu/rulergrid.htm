The Rulergrid dialog box lets you display a grid of a specific size and orientation on the surface. This lets you compare the features on the surface with a grid of known size. This can assist in planning manipulations, or in understanding the general size of objects on your sample.

In order to see the rulergrid, you must enable it.

To use the rulergrid, choose Rulergrid from the Analysis menu. This will bring up the Rulergrid Analysis window:

To turn on the rulergrid, click on the Rulergrid Enabled checkbox. The box will turn red, indicating that the rulergrid is now on, as shown in the image. 

You should now be able to see a grid overlaid on the surface.

Changing the rulergrid’s scale can give you more or less precision in your estimations. Increasing the Grid spacing value spreads out the grid lines, and decreasing the Grid spacing value concentrates the grid lines.

Changing the rulergrid’s angle can make some estimations easier. Type an angle in degrees into the Grid angle (degrees) box to rotate the grid.

Changing the rulergrid’s color can enhance the effectiveness of your rulergrid, or make your image more aesthetically appealing. To modify the rulergrid’s color, click on the button labeled Set rulergrid color.

Colors on the rulergrid are chosen using the standard Windows color selection dialog box. You can choose the color from a palette of pre-defined colors, or specify your own. 


The next two controls link the rulergrid to the Measure Lines. The Set grid offset to red line checkbox attaches one intersection of the grid to the red measure line. The Set angle to green line checkbox changes the angle of the rulergrid so one of the grid lines goes through the green line. These two controls can be used in conjunction to align the grid easily with features on the surface.

Note that you can position the red and green measure lines before you activate the Set grid offset to red line and Set angle to green line checkboxes.

The Grid X offset and Grid Y offset allow you to type in the location of one of the grid intersections.

The X and Y line width controls allow you to increase the width of the rulergrid lines, to make them more obvious. When viewing the surface at a shallow angle, it may be helpful to increase the width of just one set of grid lines for better visibility.

Finally, the Opacity control changes whether the rulergrid lines are completely solid, at 100%, or whether some of the surface color can show through. 0% opacity will make the rulergrid invisible.
