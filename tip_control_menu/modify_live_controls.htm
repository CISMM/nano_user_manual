The Modify Live controls dialog box is for controls that must be used during a modification. The Slow Line tool is the only one that requires these controls in this release of the NanoManipulator.

To use the slow line controls, first specify the path the slow-line will follow, using the following steps:

Choose the Touch phantom mode. 
Press the phantom button, touch the sample surface, determine where the slow line path will start. 
When you release the button, a blue marker line will appear. Each time you release the button, another point on the slow line path is specified, marked by a blue line.
When you are done specifying the path, hit commit to move the SPM tip to the first point and apply modification force. 

Play/Pause

When you hit the play button, its name changes to pause, and the tip steps forward, collecting data at each step. If you then hit the button again, its name changes back to play, and the tip will stop. The tip will continue collecting data at the point where it is paused.  

Step

Take one step forward, and stop. The tip will continue collecting data at the point where it is stopped. 

Forward/Reverse

Controls the direction of motion. When the commit button is pressed, the tip moves to the first marker specified, and switches to using modification parameters. The step or play buttons will move the tip along the path in the order the markers were specified if forward is chosen. The step or play buttons will move the tip along the path in the reverse order the markers were specified if reverse is chosen.

Step Size

Controls how far the tip moves if the step button is pressed, and controls the how far the tip moves before it takes another data point when the play button is pressed.
