Image Mode Parameters Dialog Box

This window allows you to control the behavior of Image Mode. The parameters control how the AFM tip behaves as it scans the surface, and how hard it presses as the user feels the sample surface in preparation for a modification.

The button at the top, labeled either Quick or Full, controls which set of image controls are displayed. 

Quick controls

These controls are designed to take effect immediately, so they can change the parameters that are used during imaging. They are "tweak" controls, so if a parameter is slightly wrong it can be adjusted easily, in a control panel which will not obscure the 3D view of the surface. 

The Image state box shows whether the SPM is using oscillating or contact imaging. 

The controls available are Setpoint, P-Gain, I-Gain, and D-gain, and have the same functions as in the ThermoMicroscopes SPMLab software and the Full Controls, described next. 

Full Controls

This dialog provides a full set of controls for imaging. They are designed to allow you to change the values of several controls, evaluate the settings, and then approve or reject the new settings with the Accept or Revert buttons.

Mode

Oscillating or Contact. These modes are the same as those provided by the Thermo software. This dialog provides an additional set of parameters so that you can make an automatic switch to a different mode atomically.

Setpoint - nA for Contact, % for Amplitude Oscillating, nA for Phase Oscillating. 
Proportional Gain - same as Thermo software.
Integral Gain - same as Thermo software.
Derivative Gain - same as Thermo software.
Rate (um/sec) - Control the maximum rate of the modification. BUG it doesn't affect the speed of modification.
Amplitude - For oscillating mode, same as Thermo software.
Frequency - For oscillating mode,  same as Thermo software.
Input Gain - For oscillating mode,  same as Thermo software.
Drive Attenuation - For oscillating mode,  same as Thermo software.
Phase or Amplitude - For oscillating mode,  same as Thermo software.
Phase angle - For oscillating mode,  same as Thermo software.

Note that not all parameters are meaningful for each kind of imaging. Contact mode imaging ignores all parameters after Rate, while Phase Angle is used only by Phase Oscillating imaging. It is unlikely that you will want to change many of the parameters used for Oscillating imaging, since the NanoManipulator does not provide the oscilloscope and graph displays that the ThermoMicroscopes software provides to assist in setting these parameters. These controls are available mainly to record the values you set up in the ThermoMicroscopes software. 

Grid Resolution

This pop-up menu changes the number of sample that are collected during scanning. It has the same values which are available in the Thermo software.

Accept and Revert

The Accept button will send your changed parameters to the SPM to be used. It will turn pink when the parameters have been changed from the current imaging parameters. The Revert button resets the controls so they show the image parameters the SPM is currently using.
